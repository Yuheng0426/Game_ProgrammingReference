# 13 Expert - Behavior Tree, Boss AI, Phase System

This folder teaches a small behavior tree for boss AI.

## What You Learn

- Build simple behavior tree nodes.
- Use selector and sequence logic.
- Change boss behavior based on health phases.
- Keep boss decisions separate from animation or rendering code.

## Why It Matters

Boss fights often outgrow basic state machines. Behavior trees make it easier to compose conditions and actions, especially when a boss has multiple phases, cooldowns, distance checks, and special attacks.
