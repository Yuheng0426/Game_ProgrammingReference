# 10 Advanced - Skill Tree, Progression, Unlock System

This folder teaches how to model an upgrade tree without hard-coding every unlock in UI code.

## What You Learn

- Store skill metadata in small data structures.
- Check level, point, and prerequisite requirements.
- Unlock skills safely through one validation path.
- Keep progression rules independent from graphics or menus.

## Why It Matters

Action RPGs, roguelites, soulslike builds, and shooter perk systems all need clear progression logic. A clean unlock path prevents balance bugs and UI-only rule mistakes.
