# 11 Expert - Gameplay Event Bus, Achievement System

This folder teaches a small event bus that lets gameplay systems talk without direct dependencies.

## What You Learn

- Create typed gameplay events.
- Subscribe systems to events.
- Broadcast player actions such as kills or item pickups.
- Unlock achievements without coupling achievement code to combat code.

## Why It Matters

Large games often become difficult to maintain because every system knows too much about every other system. Event-driven gameplay can keep combat, UI, audio, quests, analytics, and achievements cleaner.
