# 08 Beginner - Player Controller, Movement, Camera Follow

This folder teaches the first gameplay system many games need: a controllable player.

## What You Learn

- Convert input into movement.
- Smooth velocity with acceleration.
- Clamp the player inside a small playable area.
- Make a camera follow the player without snapping.

## Why It Matters

Shooter games, soulslike games, action RPGs, platformers, and top-down games all start with readable movement code. If the player controller is unclear, every later combat or camera system becomes harder to debug.
