# 09 Intermediate - Combat Hitbox, Hurtbox, Frame Data

This folder teaches a common action game combat idea: attacks only deal damage during active frames.

## What You Learn

- Define startup, active, and recovery frames.
- Enable hitboxes only during the active window.
- Check hitbox versus hurtbox overlap.
- Prevent one attack from hitting the same target more than once.

## Why It Matters

Soulslike games, fighting games, action RPGs, and melee shooter enemies all rely on clear frame data. Good frame data code makes combat easier to tune and easier to explain.
